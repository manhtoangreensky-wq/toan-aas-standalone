The nine exact frozen Bot Feedback menu values below are private static-audit source evidence for a **fresh signed** Web Support Desk navigation only. The Browser never receives the raw Bot callback or category in a route, query, fragment, form, storage, API request or support record. `Web audit intent` is a static reviewer label, not a preselected Web category and not a customer-controlled value. The customer must choose a category, describe the issue and submit a new owner-scoped Web case through the existing CSRF and idempotency boundary.
